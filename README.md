Project Overview

This project analyzes restaurant data from Zomato to identify customer preferences, restaurant trends, ratings, online ordering behavior, and pricing patterns.

Dataset Information

The dataset contains:

Restaurant Name
Online Order Availability
Table Booking Availability
Ratings
Votes
Approximate Cost for Two
Restaurant Type
Business Questions Solved
Which restaurant types receive the highest ratings?
Does online ordering impact restaurant ratings?
Which restaurants have the highest number of votes?
What is the average cost for different restaurant categories?
Does table booking influence ratings?
Which restaurant type is most common?
Tools Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
Key Insights
Most restaurants belong to the Dining category.
Restaurants offering online ordering tend to receive more customer engagement.
Higher-rated restaurants generally attract more votes.
Cost and ratings show moderate correlation.
